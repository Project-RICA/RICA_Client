RICA Instruction

사용 설명 및 초기 셋팅 방법 ℹ️ 본 창은 구글 자동로그인이 설정되어있지 않은 경우에만 나타납니다.

1. 먼저 구글 로그인을 진행해주세요.

자동로그인이 되어있어야 앞으로 RICA가 실행될 때마다 자동으로 유튜브에 접속하여 작업을 시작할 수 있습니다.

그렇지 않으면 매번 로그인을 수동으로 하셔야 합니다 😩

구글에 로그인이 되어있는 것을 RICA가 확인하면 시작 시 바로 유튜브 창으로 넘어갑니다. 이후 모든 과정은 자동으로 진행됩니다.

구글 로그인 창으로 이동 ▶️

2. 보안이 취약한 컴퓨터로 로그인 시 제한된 권한의 계정 사용을 추천합니다.

독립적 환경의 크롬이어도 크롬이긴 하므로 보안 수준은 크롬과 비슷할 테지만 아래 ℹ️에 해당되지 않으시면 보안을 위해 채널 관리자 계정 대신 댓글관리자 역할을 가지고 있는 계정을 사용하세요.

ℹ️ 컴퓨터가 백신으로 단단히 무장했다거나 계정에 2단계 인증을 걸어놓으신 경우 관리자 계정으로 로그인해도 괜찮습니다.

3. 이 크롬은 여러분이 쓰시는 크롬과 독립된 환경에서 실행되는 크롬입니다.

문제 해결을 위한 크롬설정 등은 현재 이 크롬으로 해주시기 바라며 본인이 기존에 쓰던 크롬에서 설정하는 것은 아무런 의미가 없습니다.

원활한 크롬 제어를 위해 팝업 항상 허용 옵션을 꼭 켜주시기 바랍니다. 현재 개발을 팝업 모두 허용한 상태에서 진행하고 있기 때문에 Selenium 에서 JS로 보내는 "새창열기" 명령이 먹히지 않을 경우 문제가 생길 수도 있습니다. 안 킨다고 바로 문제가 생기는 것은 아니지만 추후에 문제가 발생할 가능성이 매우 크기에 처음 설정 시 설정해두시는 것을 추천드립니다.

ℹ️ 크롬의 보안정책 강화로 인하여 개발자가 직접 popup 옵션을 수정하는 것이 불가능 하기 때문에 유저가 직접 수정해야 하는 것입니다.
⚠️ RICA는 오직 독립적으로 실행되는 크롬에만 접근합니다. 만약 자신이 기존에 쓰고 있던 크롬이 열리는 등의 상황이 발생하면 즉시 프로그램을 종료하고 RICA 공식 GitHub에서 받은 파일이 맞는지 확인하세요. 계정 해킹의 위험이 있을 수도 있습니다. 😨
